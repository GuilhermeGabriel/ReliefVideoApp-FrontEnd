## Installation

```bash
# node v18.16.0
# npm 9.5.1

npm install
```

## Running the app, open [http://localhost:4200](http://localhost:4200) to view it in the browser.

```bash
# start 
npm run start

# to generate production mode
npm run build
```

Technologies used:
- ReactJs
- MaterialUI
- React Context
- Axios
